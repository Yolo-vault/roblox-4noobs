 Este e um guia de estudos criado para ajudar iniciantes a aprenderem scripting em Lua, especialmente no contexto de desenvolvimento para Roblox. Este guia foi elaborado com base na minha própria jornada de estudos, e espero que ele possa te ajudar a dominar os fundamentos de scripting em Lua.

## Sobre o Projeto

 Este guia e estruturado em módulos que cobrem desde os conceitos básicos ate
 tópicos mais avançados. Cada modulo contem aulas detalhadas, exemplos práticos, quizzes e tarefas para reforçar o aprendizado.
